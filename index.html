Exportar Backup
Download completo em JSON

Gera um arquivo JSON com todos os dados do sistema: movimentações, devoluções de embalagens, análises, usuários e dados mestres. Ideal para backup de segurança.

Importar Backup
Restaurar a partir de arquivo

Restaura o sistema a partir de um backup JSON exportado anteriormente. Atenção: todos os dados atuais serão substituídos pelos dados do backup.

Recomendamos exportar um backup dos dados atuais antes de importar. Esta ação não pode ser desfeita.
